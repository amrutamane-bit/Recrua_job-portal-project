# 💼 Recrua - Job Portal Web Application

## 📌 Project Overview
Recrua is a **cloud-based Job Portal Web Application** that allows users to register, login, search for jobs, and apply online. It demonstrates full-stack development using frontend, backend, and database integration.

---

## 🚀 Features

- 🔐 User Authentication (Signup & Login)
- 🔍 Job Search by Category and Location
- 📄 Apply for Jobs
- 🕒 Application History Tracking
- ☁️ Cloud Deployment
- 💾 Data stored in MongoDB database

---

## 🧑‍💻 How to Run Locally

1. Clone the repository:
  git clone https://github.com/amrutamane-bit/Recrua_job-portal-project.git

2. Navigate to project folder:
   cd job_portal

3. Install dependencies:
   npm install

4. Start server:
   node server.js

5. Open browser:
   http://localhost:3000/login
